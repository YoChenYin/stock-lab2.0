# Stock Lab 2.0 操作手冊

---

## 一、平台簡介

Stock Lab 2.0 是一套雙市場選股分析平台，整合台股（FinMind 法人籌碼）與美股（SEC/FINRA/yfinance 多源數據），每日排程自動更新資料，提供進場、出場、族群輪動三大功能。

**網址結構**

| 路徑 | 功能 |
|------|------|
| `/tw` | 台股選股主頁 |
| `/us` | 美股籌碼雷達主頁 |

---

## 二、台股頁面（/tw）

### 2-1 頁面結構

進入 `/tw` 後自動載入四個區塊：

```
┌─────────────────────────────────────┐
│  📊 因子熱度榜（Factor MAX）         │  ← 預設展開
│  🔥 族群資金熱圖                     │  ← 預設展開
│  ┌──────────────────────────────┐   │
│  │ 全部 │ 進場選股 │ 出場警示   │   │  ← 三個 Tab
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘
```

---

### 2-2 全市場掃描（全部 Tab）

**顯示所有 82 支台股**，每支標記三種狀態：

| 狀態 | 判斷條件 |
|------|----------|
| 📈 進場 | 有正向信號（翻轉↑ / 雙強 / 蓄力）或 SMS ≥ 55，且出場警示 ≤ 1 個 |
| 🚨 出場 | 同時觸發 ≥ 2 個出場警示 |
| ⚪ 觀望 | 不符合以上條件 |

**欄位說明**

| 欄位 | 說明 |
|------|------|
| 股票名稱 | 點擊展開個股詳情（法說紀錄 + 策略快覽） |
| 股價 / 漲跌 | 即時抓取 |
| 近三月 YoY | 月營收年增率，連續 3 個月 |
| 集中度 5 日 / 趨勢 | 法人 5 日淨買超 / 成交量，正數代表資金流入 |
| 趨勢加速 (Δ) | 集中度 5 日 vs 20 日差值，> 0 代表加速買入 |
| 外資連買 / 連賣 | 正整數 = 連續買超 N 天；負整數 = 連續賣超 |
| 投信連買 / 連賣 | 同上 |
| 標籤 | 見下方「信號標籤」說明 |

**信號標籤速查**

| 標籤 | 意義 |
|------|------|
| 🔥 雙強 | 月營收連 3 月成長 + 5D 法人集中度 > 10D > 20D（全面加速） |
| ↑ 翻轉 | 外資或投信近 3 日由賣轉買 |
| 🗜 蓄力 | 近 20 根 K 線振幅收縮，等待方向 |
| 📈 營收↑ | 近 3 月 YoY 均正且最新月加速 |
| 👥 籌碼↑ | 5D 集中度 > 10D > 20D（但未達雙強） |
| ⚠️ 營收衰退 | 最新月 YoY 為負（注意，非淘汰條件） |

---

### 2-3 進場選股（進場 Tab）

**已過濾**，只顯示觸發任一進場條件的股票，自動排除：

- 空頭排列（MA5 < MA10 < MA20）
- 籌碼加速流出（5D 比 20D 低超過 1%）
- 雙月 YoY 均為負

**排序**：雙強優先 → 集中度加速 (Δ) 降序

**如何選股**

1. 優先看「雙強」標籤股票 — 基本面 + 籌碼雙重確認
2. 確認「趨勢加速 (Δ)」為正（資金加速流入）
3. 查看「外資成本」— 現價低於外資成本代表法人被套、反彈動力強
4. 點擊股票名稱 → 確認 52 週位置（低位更佳）

---

### 2-4 出場警示（出場 Tab）

**任一條件觸發即列入**，依警示數量排序：

| 警示 | 說明 |
|------|------|
| 🔴 外資轉賣 / 投信轉賣 | 近 3 日由買超翻轉為賣超 |
| 🔴 法人同步賣超 | 外資 + 投信同時連賣 ≥ 3 天 |
| ⚠️ 跌破五日線 | 收盤低於 MA5 |
| ⚠️ 空頭排列 | MA5 < MA10 < MA20 |
| 📉 籌碼流出 | 5D 集中度比 20D 低超過 1% |
| 💸 營收衰退 | 近兩月 YoY 均負 |

**嚴重等級**：🔴 高風險（≥ 3 個）｜🟠 中風險（2 個）｜🟡 低風險（1 個）

---

### 2-5 族群資金熱圖

顯示 13 個產業族群的法人資金強度，用長條顯示資金流向：

- **綠色長條** → 資金流入（5D 法人集中度 > 0）
- **紅色長條** → 資金流出
- 點擊族群名稱 → 切換個股明細卡片

**操作建議**：選擇綠色 + 長條最長的族群優先，再進入進場 Tab 找個股。

---

### 2-6 個股詳情（點擊股票名稱）

展開後顯示兩區塊：

1. **最近法說紀錄**（MOPS）— 日期 + 事件標題 + PDF 連結
2. **策略快覽**
   - 52 週位置橫條（一眼知道當前價格在高低點哪裡）
   - 外資成本（60 日加權均成本，只含買超日）
   - 與現價差距（負值代表外資被套）
   - SMS 籌碼分數（0–100）

## 三、美股頁面（/us）

### 3-1 頁面結構

```
┌───────────────────────────────────────┐
│  資料更新時間 ● 2026-04-30 23:45      │
│  VIX / 10Y 殖利率 / Mag7 / 板塊輪動   │  ← 市場環境卡片
│  📊 因子熱度榜（Factor MAX）           │  ← 預設展開
│  ┌──────────────────────────────────┐ │
│  │ 全部 │ 進場選股 │ 出場警示       │ │
│  └──────────────────────────────────┘ │
└───────────────────────────────────────┘
```

---

### 3-2 市場環境卡片（頁面頂部）

| 卡片 | 說明 | 操作意義 |
|------|------|----------|
| 😱 VIX | 當前值 + 5 日均 + 風險等級 | VIX > 25 → 進場分數自動打 8 折 |
| 🏦 10Y 殖利率 | 當前值 + 20 日斜率 | 快速上升 → 成長股減分 |
| 🔭 Mag7 乖離 | 七巨頭相對 50MA 平均乖離 | > 15% 警示 → AI 板塊過熱 |
| 🔄 板塊輪動 | XLK / XLF / XLP 單日漲跌 | 判斷資金在科技 / 金融 / 防禦哪個板塊 |

---

### 3-3 進場選股（進場 Tab）

**選擇週期（右上角下拉選單）**

| 週期 | 適合族群 |
|------|----------|
| 短線（1–5 天） | 事件驅動、消息面操作 |
| 波段（1–4 週）（預設） | 多數波段交易者 |
| 中線（1–3 月） | 長線持有者 |

**籌碼評分五維度（各 0–100 分）**

| 維度 | 資料來源 | 含義 |
|------|----------|------|
| 👤 內部人 | SEC Form 4 近 90 天 | 管理層買賣方向 |
| 📉 空頭 | FINRA 空頭興趣 | 空頭比例趨勢（高分 = 空頭回補） |
| 📊 量能 | OBV / CMF / MFI | 資金持續流入 |
| 🎯 選擇權 | 買權 / 賣權流量 | 機構籌碼方向 |
| 🏛 機構 | 13F 持倉季度變化 | 大型機構加減倉 |

**標籤說明**

| 標籤 | 觸發條件 |
|------|----------|
| 🐋 巨鯨 | 量能 + 機構雙重異常（whale_alert） |
| ⚡ 進場 | 技術面多重信號共振（entry_timing） |
| insider↑ | 近期多位內部人買入（Form 4，自動置頂） |
| options 異常 | 買權流量 Z-score 顯著偏高 |
| 🔥 因子爆發 | 所屬族群 5 日 MAX ≥ 2% |
| ⚠ 個股過熱 | 單日漲幅 > 5%（短線反轉風險） |

---

### 3-4 全市場掃描（全部 Tab）

顯示全宇宙股票，**有 insider↑ 的股票自動置頂**，依綜合分排序。

| 狀態 | 條件 |
|------|------|
| 📈 進場 | 綜合分 ≥ 65，或觸發 entry_timing，或巨鯨且分數 ≥ 50 |
| 🚨 出場 | 綜合分 < 35，或技術止損觸發 |
| ⚪ 觀望 | 其他 |

---

### 3-5 出場警示（出場 Tab）

**籌碼類警示**

| 警示 | 觸發條件 |
|------|----------|
| 綜合分偏低 | composite < 35 |
| 空頭率高 | short_score < 30 |
| 量能派發 | volume_score < 30 或 OBV 持續背離 |
| 機構減倉 | institutional_score < 30 |

**技術類警示**

| 警示 | 觸發條件 |
|------|----------|
| 🛑 硬性止損 | 距 20 日高點下跌 > 7% |
| 跌破 20MA × 3 日 | 連續 3 天收盤低於 20MA |
| RSI 頂背離 | RSI > 75 後下滑，或價格新高但 RSI 不創新高 |
| ATR 止盈線破 | 低於最近高點 - ATR(14) × 2 |

> ⚠️ 注意：**內部人賣出（insider↓）不列為出場條件**。高管賣股常為稅務 / 多元化需求，與股票前景無直接關係。

---

### 3-6 個股詳情（點擊股票名稱）

展開後顯示四個區塊：

1. **Form 4 內部人交易**（近 90 天，前 5 筆）— 日期、職銜、買賣股數與金額
2. **選擇權流量快照** — 買 / 賣權比、OTM 買權比率
3. **13D/13G 大戶申報** — 主動持股（🔴）/ 被動持股（🔵）
4. **技術指標面板**
   - 進場分數（0–100，含 VIX 折扣）
   - 出場風險（0–100）
   - 六項進場技術信號：MA 對齊 / RSI 低點翻轉 / MACD 金叉 / 布林突破 / 雙底 / VCP 縮量
   - 四項出場技術信號：硬止損 / 跌破 20MA / RSI 背離 / ATR 止損

## 四、因子熱度榜（Factor MAX）

台美兩頁面頂部皆有，顯示**因子組合**（不是個股）的近 20 日最大單日報酬：

| 欄位 | 說明 |
|------|------|
| 因子名稱 | 如「AI 算力」「外資買超」「SaaS」 |
| MAX 報酬 | 近 20 個交易日最大單日等權報酬 |
| 發生時間 | MAX 出現的日期 |
| 20 日均報 | 近期平均每日報酬 |
| 動能分（0–100） | MAX 強度 × 時間近遠加權 |
| MAX 當日領漲股 | 那天漲最多的 Top 3 個股 |
| ⚠️ 個股過熱 | 個股單日漲 > 5%（樂透效應，短線注意反轉） |

**使用邏輯**：因子層面的高 MAX 代表系統性動能；個股層面的高 MAX（單一股票爆漲）則要小心反轉。

## 六、目前平台限制

### 功能限制

| 限制 | 說明 |
|------|------|
| 🔕 無警示通知 | 無法在股票觸發條件時主動推送通知 |
| 📋 無自選股清單 | 無法儲存個人追蹤股票 |
| 🔒 無登入驗證 | 所有人可訪問，無個人設定 |
| 📤 無匯出功能 | 無法匯出 CSV / Excel |
| 📊 無回測介面 | 無法驗證信號歷史有效性 |

### 資料限制

| 限制 | 原因 | 影響 |
|------|------|------|
| 美股內部人交易無資料 | EDGAR API 解析問題 | insider_score 偏低，insider↑ 標籤無法正確觸發 |
| 台股僅 82 支 | 手動維護 sector_data.py | 非追蹤清單內的股票無法掃描 |
| 美股宇宙需每月重建 | build_universe.py 需手動執行 | 新上市股票不自動加入 |
| 法說資料依公司公告 | MOPS 無固定頻率 | 可能顯示數週前的法說 |
| 機構持倉季度更新 | 13F 每季申報 | institutional_score 最舊可能 3 個月前 |
| 空頭興趣半月更新 | FINRA 每兩週公布 | short_score 有滯後 |

### 技術限制

| 限制 | 說明 |
|------|------|
| 單一 SQLite | 高併發寫入有鎖定風險（單一 pod 目前無影響） |
| yfinance 依賴 | 非官方 API，偶有 rate limit 或格式變更導致抓取失敗 |
| FinMind 免費層 | 每秒約 2 requests，預抓 80+ 支需約 40 秒 |
| 記憶體快取 | 重啟後 TTL cache 清空，首次請求需重新計算 |

## 七、快速操作流程

**找台股進場機會**
```
/tw → 進場選股 Tab → 優先看「雙強」→ 確認集中度 Δ 為正 → 點擊看法說 + 52 週位置
```

**監控台股出場訊號**
```
/tw → 出場警示 Tab → 紅底（≥ 3 個警示）優先處理 → 確認空頭排列 + 跌破五日線
```

**找美股強勢籌碼**
```
/us → 進場選股 → 選週期 → 看 insider↑ 置頂股 → 點擊看 Form 4 明細 + 選擇權流量
```

**判斷大盤氛圍**
```
/us 頁面頂部卡片 → VIX > 25 縮部位 → Mag7 乖離 > 15% 避免追高 → 板塊輪動看資金偏好
```
